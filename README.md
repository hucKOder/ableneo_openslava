# ableneo_openslava
Workspace for the Hands on lab.

__Installation__

Setup python environment:
```
python -m venv name_of_env
```
Run environment:

- Win
```
.\name_of_env\Scripts\activate.bat
```
- Ubuntu/Mac OS
```
source name_of_env/bin/activate
```
Install packages:
```
pip install -r requirements.txt
```
